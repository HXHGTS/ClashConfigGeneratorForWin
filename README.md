# ClashConfigGeneratorForWin

自己搞的windows版Clash配置编辑工具，能用就行

基于[Loyalsoldier](https://github.com/Loyalsoldier/clash-rules)提供的规则列表

暂时单次只能导入一个节点，且暂时不支持Shadowsocks(自己很少用)，后期慢慢改
